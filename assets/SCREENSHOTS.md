# Screenshots guide

Store the plugin screenshots in `assets/screenshots/` with the following filenames so they map 1:1 with WordPress.org expectations:

1. screenshot-1.png — Setup screen (Settings)
2. screenshot-2.png — Import screen (JSON import)
3. screenshot-3.png — Logs and statistics

Recommended size: 1200×900 px (PNG, optimized). Keep UI readable in dark/light.

Update README.txt > Screenshots captions if needed.
